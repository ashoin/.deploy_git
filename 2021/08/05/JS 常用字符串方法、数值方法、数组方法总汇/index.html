JS 常用字符串方法、数值方法、数组方法总汇

JavaScript 字符串方法

length 属性

返回字符串的长度

split() 方法

将字符串转换为数组

search() 方法

该方法设置一个参数：需要搜索的字符串。

搜索特定的字符串，并返回位置

slice() 方法

该方法设置两个参数：起始索引（开始位置），终止索引（结束位置）。[ 左闭右开 )

提取参数对应位置的字符，并返回新的字符串。

substr() 方法

该方法设置两个参数：起始索引（开始位置），提取字符的长度。

提取参数对应位置及长度的字符，并返回新的字符串。

replace() 方法

该方法设置两个参数：需要被替换的字符串，替换成新的字符串。

不改变原来的字符串，返回新的字符串。

可以通过正则表达式，扩展替换的功能。

concat() 方法

连接连个或者多个字符串；

trim() 方法

删除字符串两端的空白符。

JavaScript 数值方法

toString() 方法

将数字转换成字符串。

toFixed() 方法

返回一个指定位数小数的字符串。

Number() 方法

将变量转换为数值。

JavaScript 数组方法

join() 方法

该方法设置一个参数：数组中元素以什么方式连接。

将数组元素结合为一个字符串。

pop() 方法

从原数组中删除最后一个元素。

push() 方法

在原数组结尾处添加一个新的元素。

shift() 方法

删除原数组的首个元素。

unshift() 方法

在原数组首位添加一个新的元素。

splice() 方法

该方法有多个参数：第一个参数定义添加新元素和删除旧元素的位置；第二个参数定义删除元素的位数；其余参数皆为要添加到数组中的元素。

返回一个包含一删除项的数组
